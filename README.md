# 🚢 Mini TOS (Terminal Operating System)

## 1. 프로젝트 개요

### 프로젝트명

**항만 터미널 운영 시스템 (Mini TOS)**

### 프로젝트 소개

항만 터미널에서 발생하는 선박 입출항, 컨테이너 적치, 게이트 입출고 등의 업무를 관리하는 TOS(Terminal Operating System)를 단순화하여 구현한다.

컨테이너 등록, 상태 변경, 이벤트 이력 관리 기능을 중심으로 시작하여 야드, 게이트, 선박 관리 기능을 단계적으로 확장한다.

## 2. 프로젝트 목표

실제 항만 운영 프로세스를 기반으로 컨테이너의 위치 추적 및 작업 이력을 관리할 수 있는 시스템을 구축하는 것과 TOS의 핵심 도메인을 단계적으로 구현하면서 항만 물류 운영 프로세스와 시스템 구조를 이해하는 것을 목표로 한다.

### 세부 목표
컨테이너 마스터 데이터 관리
컨테이너 상태 변경 및 추적
작업 이벤트 이력 관리
야드 적치 위치 관리
게이트 입출고 관리
선박 입출항 관리
운영 현황 모니터링 및 집계 정보 제공
